# Customer Member Card Logo Design

## Goal

Apply the approved POS GLASSES Optic Bridge logo to the customer CR80 membership card shown in the preview modal and to the separate print-window document generated by `printMemberCard`.

## Current State

The card currently renders a temporary `PG` text mark plus separate brand-name and tagline text. The preview and print document duplicate this markup and their CSS independently. Barcode rendering, member identity fields, and CR80 dimensions already work and must remain unchanged.

## Selected Layout

Replace the complete temporary brand cluster with the existing horizontal logo asset at `/assets/images/pos-glasses-optic-bridge-logo.png`. Place it inside a compact warm-white logo plate in the upper-left area of the dark card. The plate gives the teal and deep-pine artwork sufficient contrast on screen and on printed CR80 media.

The preview uses a 142 × 42 px plate. The print document uses a proportional 36 × 10 mm plate. Both use `object-fit: cover` and centered positioning to crop the square source asset around its horizontal lockup. The existing `Member` badge remains on the right.

## Behavior and Data

Only brand markup and styling change. Customer name, phone, tier, EAN-13 barcode, member code, print-window creation, and automatic `window.print()` behavior remain intact. The logo uses descriptive `alt="POS GLASSES"` text in both preview and print HTML.

## Validation

Add a static regression test that verifies `customers.html` references the final logo asset in both the preview builder and print template, contains the new preview and print logo styles, and no longer renders `STORE_BRAND.mark` as a temporary card mark. Run the focused frontend test and the complete test suite.
